About Me

My name is Tiffany. I am excited about web development and looking for to get started as a Front End Developer with the hopes of transitioning into a Full Stack Developer. I am interested in HTML/HTML5, CSS/CSS3, Javascript and PHP. I enjoy finding out how websites work and improving websites to make them more functional for the user. I graduated with a degree in Management Information Systems where I took courses over PHP, HTML, ASP.Net as well as project management. In early 2016, I decided to concentrate on improving my developer skills so I left my job as a Data Integrity Analyst to join Coding Dojo in Dallas. At Coding Dojo, I learned 3 full stacks in three months - LAMP, MEAN and Rails. While, there I completed many projects including a Travel Site using Codeigniter using the LAMP stack (code for that site can be found here) and a Store similar to iTunes in the MEAN stack (code for that site can be found here). Travel Plan PHP - Mac OS X, Apache web server, MySQL database, PHP: CodeIgniter

This project on github: Travel Plan PHP

For this project, a user can register or login into the dashboard. Once logged in, the user can add a travel plan - selecting a destination, description, start and end date. Once the travel plan is added, the user can see the newly added plan on a table along with any other plans they've added. In another table on the dashboard, they can see any other trips other users have planned and also have the option to join them on their trip. If they join, this plan will be added to the table of trips they are taking. When the user is done, they are able to log out.
